新冠援助平台

新冠病毒疫情中的援助信息网站

官方网站

源码仓库

关键词
武汉 疫情 援助 PWA WebCell

项目概况

2019-nCov 疫情在 2020 年春节期间爆发，不但重灾区武汉物资告急，全国各地也捉襟见肘，急需各项专业信息的大规模流通。

于是，开源社官网组组长水歌基于自己开源的 WebCell 框架，在开源教育组赵生宇、王伟和成员发展组林张炜发起的 wuhan2020 数据仓库的基础上，一晚上便开发出一个 Web 应用的雏形，并带领 20 多位志愿者在十多天内完成前后端的开发。

自平台第一个模块（医疗物资需求）上线开始，在未做专门宣传的情况下，日 API 访问量峰值超 3 万次，武汉、湖北之外的各省市也提交了很多数据，说明全国疫情之危急。

基本用法

谷歌 Chrome、谋智 Firefox、欧朋、微软 Edge、UC 浏览器、小米浏览器、QQ 浏览器等均可安装本网站到桌面（Safari 对 W3C 国际标准支持迟缓，只能添加图标，网络缓存、离线启动等高级功能可能无法使用）
